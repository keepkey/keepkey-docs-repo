## Hardware Requirements

### Supported Operating Systems
- Windows 10 or later (64-bit)
- macOS 10.15 (Catalina) or later
- Linux: Ubuntu 18.04 or later, or equivalent

### Hardware Specifications
- USB 2.0 port or higher
- Internet connection
- Minimum 1GB RAM
- 500MB available storage

## Software Requirements

### KeepKey Desktop Application
- **Platform Support:** Desktop only
- **Mobile Support:** Not available
- **Operating Systems:** Windows, macOS, Linux
- **Type:** Standalone application

### Browser Extension
- **Current Support:** Google Chrome only
- **Future Support:** Additional browsers coming soon
- **Minimum Version:** Chrome 88 or later

## Network Requirements
- Stable internet connection required
- Minimum 1Mbps download speed
- Recommended 5Mbps or higher

## Important Notes

**Device Firmware:**
- Latest firmware version recommended
- Automatic updates available through KeepKey Desktop

**Platform Limitations:**
- No mobile device support
- Desktop platforms only
- Chrome-exclusive for browser extension

> **Note:** For optimal security and performance, always keep all software components updated to their latest versions.
